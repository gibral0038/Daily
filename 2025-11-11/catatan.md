- Pada kode ini menggunakan sebuah fungsi peniympanan dinamis yang terdapat dalam c++ yaitu "unordered_map" dimana kegunaan dari fungsi penyimpanan dinamis ini merupakan seubah tipe data buatan untuk menyimpan 2 buah tipe data dengan format <key, value>.

- Penggunaan dari unordered_map yaitu untuk menyimpan sebuah data sebagai kunci dimana ketika data tersebut dipanggil maka akan memunculkan nilai sebagai representasi dari data tersebut atau bisa disebut bahwa data yang pertama merupakan inisial untuk memanggil data kedua.
